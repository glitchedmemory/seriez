2026-06-04

## 1. Letterboxd

**핵심 기능**
- 영화 시청 기록 (Diary)
- 별점 평가 (5점 + 0.5점)
- 리뷰 작성 및 공유
- 리스트 (Top 500, 장르별, 연도별 등)
- 위시리스트 (Watchlist)
- 친구 팔로우 및 활동 피드
- 저널 (영화 관련 블로그/칼럼)

**UI/UX**
- 깔끔한 미니멀 디자인, 어두운 테마
- 영화 포스터 중심의 그리드 레이아웃
- 상단 네비게이션: Films / Lists / Members / Journal
- 모바일 앱(iOS/Android)에서도 일관된 경험

**강점**
- 영화 덕후 커뮤니티가 압도적으로 활발
- 리뷰 문화가 잘 형성됨 (짧고 재치있는 리뷰 중심)
- 유머 감각 (Backrooms 영화 리뷰: "white boys are wearing crop tops in horror movies again god bless" → 39,457 좋아요)
- 36억 건의 영화 시청 기록

**약점**
- TV쇼, 애니메이션 추적 불가 (영화 전용)
- Pro($19/yr) 없으면 통계/스트리밍 알림 등 핵심 기능 제한
- 추천 기능이 부실 (인기 영화 나열 수준)
- 한국어 지원 없음

**수익 모델**
- 무료: 광고 포함, 기본 기능
- Pro $19/yr: 광고 제거, 통계, 스트리밍 알림
- Patron $49/yr: 커스텀 포스터, 얼리 액세스

## 2. Trakt

**핵심 기능**
- 영화 + TV쇼 통합 추적
- 스크로블 (자동 시청 기록)
- 에피소드별 체크인
- 캘린더 (방영 일정)
- 리스트 및 위시리스트
- 평점 및 리뷰

**VIP 기능 ($5/mo)**
- 연간/월간 통계 (Year in Review)
- 스트리밍 자동 연동 (Netflix, Prime, Disney+ 등)
- Plex Sync
- 고급 필터링
- 무제한 리스트 (무료는 100개 제한)

**강점**
- TV쇼+영화 통합이 잘 되어 있음
- 스트리밍 서비스 연동 자동화
- Plex 연동 (홈서버 유저에게 강점)

**약점**
- 무료 버전은 매우 제한적 — 대부분 기능이 VIP 필요
- UI가 2010년대 스타일로 구식
- Reddit에서 "버그 많고 업데이트 느리다"는 불만
- 애니메이션 DB 부실 — Simkl로 이탈하는 유저 다수

---

## 3. MyAnimeList (MAL)

**핵심 기능**
- 애니·만화 시청/독서 기록
- 별점 평가 (10점)
- 리뷰
- 시즌별 애니 목록
- 커뮤니티 포럼 + 클럽
- 애니 추천 (유저 기반)

**강점**
- 세계 최대 애니 DB (등록 작품 수 1위)
- 커뮤니티가 가장 활발
- 시즌별 신작 정보 빠름

**약점**
- **UI가 2000년대 초반 수준** — 가장 큰 불만
- 모바일 웹 경험 열악
- API가 제한적이고 레이트 리미트 심함
- 광고가 많음 (무료)
- 영어권 위주, 비영어권 지원 부족

---

## 4. AniList

**핵심 기능**
- 애니·만화 추적
- 커스텀 평점 시스템
- 통계 대시보드 (장르별, 성우별, 연도별)
- 리뷰 + 추천
- 커뮤니티 포럼
- 다크 모드 기본

**강점**
- **모던하고 깔끔한 UI** — MAL 대비 압승
- GraphQL API 공개 (개발자 친화적)
- 커스터마이징 옵션 풍부
- iOS/Android/macOS/Windows 네이티브 앱 있음
- 통계가 시각적으로 뛰어남

**약점**
- 영화·TV쇼 없음 (애니·만화 전용)
- MAL 대비 커뮤니티 규모 작음
- 한국어 지원 없음

---

## 5. Simkl

**핵심 기능**
- 영화 + TV쇼 + 애니 3종 통합 추적
- AI 기반 추천
- 자동 스크로블 (Netflix, Crunchyroll 등)
- 21개 이상 서비스에서 데이터 임포트
- iCal 연동 캘린더
- 디스코드 연동

**강점**
- **유일한 진정한 올인원** — 영화+TV+애니를 한 곳에서
- AI 추천 (시청 이력 기반)
- 데이터 이전이 쉬움 (Trakt, MAL, AniList 등 21개 서비스 지원)
- 개발자가 커뮤니티 피드백에 적극적
- API 문서화 잘 되어 있음

**약점**
- 브랜드 인지도 매우 낮음
- 유저 수 적어 커뮤니티 약함
- 무료 제한은 없지만 Pro($3/mo)에 통계 고급 기능
- UI는 깔끔하나 Letterboxd에는 못 미침

---

## 6. TV Time

**핵심 기능**
- TV쇼 시청 추적
- 에피소드 체크인
- 리액션 (이모지)
- 트렌딩 쇼
- 커뮤니티

**강점**
- TV쇼 팬덤 커뮤니티 최대 규모
- 리액션 기능이 독특 (에피소드별 이모지 반응)
- 실시간 트렌딩

**약점**
- 영화 추적 부실
- 애니메이션 지원 약함
- 추천 기능 거의 없음
- 과도한 알림

## 시장 갭 분석 (블루오션)

현재 시장에서 **비어 있는 포지션**:

1. **통합 + 모던 UI** — Letterboxd(영화·UI 굿) + AniList(애니·UI 굿)를 하나로 합친 서비스가 없음. Simkl이 시도 중이나 UI가 Letterboxd급은 아님
2. **진정한 추천 엔진** — 대부분의 서비스가 인기작 나열이나 유저 평점 순일 뿐, 개인화 추천이 약함
3. **크로스 카테고리 추천** — "이 영화 좋아하면 이 애니도 볼 만해요" 같은 크로스 추천 서비스 전무
4. **PWA 퍼스트** — 네이티브 앱 없이도 웹+모바일 동시 지원. Simkl만 앱 있음 (나머지는 별도 개발)
5. **글로벌 다국어** — 한국어 포함 아시아 언어 지원 거의 없음

## 우리 서비스 차별화 전략

1. **영화 + TV + 애니 3종 통합** — Letterboxd + Trakt + AniList를 하나로
2. **Letterboxd 수준의 UI/UX** — 포스터 중심, 심플하고 감성적인 디자인
3. **장르·감독 기반 추천 엔진** — TMDB API + 자체 알고리즘
4. **PWA로 웹+모바일 동시 커버** — 초기부터 크로스 플랫폼
5. **무료 베이스 + 프리미엄 통계/추천** — Letterboxd Pro 모델 참고
