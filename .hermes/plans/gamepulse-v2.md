# GamePulse — 기획서 v2 ("구글 검색을 대체하는 스포츠 앱")

## 1. 컨셉

**"경기 보는 사람들이 구글에서 검색하는 모든 정보를 한 앱에."**

경기 중 사람들이 구글링하는 모든 것:
점수·누가 골 넣었나·부상·베팅 오즈·실시간 뉴스·선수 스탯·중계 채널·하이라이트·판타지 영향...

→ 이걸 전부 한 앱에서 검색 없이 바로 보여준다.

## 3. 현실적인 판단

### 솔직한 답변: **기능만으로는 이기기 어렵다.**

이유:
1. 모든 기능이 **이미 무료 앱에 다 있음**
2. 사용자들은 **이미 theScore나 ESPN에 습관화됨**
3. 승부는 **기능 싸움이 아니라 데이터 싸움** — Sportradar(월 $10,000+) 같은 초저지연 데이터를 가진 쪽이 이김
4. 신규 진입 시 데이터 API 비용만 **월 $199~$300/스포츠**

### 하지만 — 진짜 틈새는 있다

위 앱들의 약점: **모든 걸 보여주지만, 사용자가 진짜 원하는 것만 골라서 보여주지는 않음.**

theScore를 열면: 뉴스 피드 → 스크롤 → 점수 탭 → 경기 선택 → 스탯 탭. **4~5단계** 거쳐야 정보 도달.

## 4. 차별화 가능한 포지셔닝 (3가지)

### 옵션 A: "1화면 = 1경기" — 초극단 단순화

**한 화면에 한 경기의 모든 핵심 정보를 담는다.**

```
┌─────────────────────────┐
│ 🏀 NYK 102 — 98 SAS  Q4 │
│                         │
│ 🔥 Brunson 34pts 8ast  │
│ 🔥 Wemby 28pts 12reb   │
│                         │
│ ❓ Is Brunson playing?  │
│ → Yes. 38min tonight   │
│                         │
│ 📊 Betting: NYK -3.5   │
│ 💰 Opened -1.5 → Now -3│
│                         │
│ 🏥 Injuries: None      │
│                         │
│ 📺 ESPN / YouTube TV   │
│                         │
│ 📰 "Knicks extend lead │
│    with 12-0 run" —ESPN│
│                         │
│ ← Swipe for next game  │
└─────────────────────────┘
```

**차별점**: 기존 앱들은 4~5탭 왔다갔다. 이건 **한 화면. 스와이프로 경기 전환.**

### 옵션 B: "AI 질문 기반" — 사람들이 실제로 궁금해하는 것만

경기 화면에 정보를 나열하는 게 아니라, **팬들이 구글에 타이핑하는 질문에 직접 답하는 UI.**

```
🏀 NYK vs SAS

🤔 "Brunson 오늘 잘하고 있어?"
→ 34pts (12-18 FG), 시즌 평균 28.3pts 대비 +20%

🤔 "Wembanyama 블록 몇 개야?"
→ 4블록, 평균 3.1개 대비 ↑

🤔 "왜 Spurs가 지고 있어?"
→ 3점슛 6-22 (27%), 턴오버 18개

🤔 "베팅 라인 어때?"
→ NYK -3.5, 오프닝 -1.5에서 상승
```

**차별점**: 메뉴·탭 없음. **질문 터치 → 즉시 답변.** AI로 자연어 답변 생성.

### 옵션 C: "게임데이 커맨드 센터" — 멀티게임 모니터링

NFL Sunday나 NBA Playoffs 같이 여러 경기 동시에 볼 때:

```
┌────────────────────────────┐
│ 🏈 1:00 PM EST            │
│ ┌──────┬──────┬──────┐    │
│ │KC 24 │BUF 17│DAL 14│    │
│ │BAL 21│NYJ 10│PHI 28│    │
│ │  Q3  │  Q4  │  Q2  │    │
│ └──────┴──────┴──────┘    │
│                           │
│ 🔴 Mahomes 310yds 3TD    │
│ 🟡 BUF RB injured (Q)    │
│ 🔴 PHI -5.5 line moving  │
└────────────────────────────┘
```

**차별점**: 기존 앱은 한 번에 한 경기만. 여러 경기 핵심 정보를 **한 화면 그리드**.

## 6. 정직한 최종 판단

### 이 앱이 성공하려면:

1. **"theScore보다 낫다"가 아니라 "theScore와 완전히 다르다"** 여야 함
2. 기능 리스트로 승부하면 **절대 못 이김** (그들은 수백명 엔지니어 + 수십억 예산)
3. **UI/UX 하나로 승부** — 질문 기반이든, 1화면이든, 멀티게임 그리드든
4. **마케팅이 전부** — Reddit r/nba, r/nfl에서 "theScore보다 빠르다" 입소문

### 솔직히:

- "검색 대체 앱"이라는 아이디어 자체는 **이미 theScore와 ESPN이 하고 있음**
- 그들이 못 하는 건 **UI 혁신** (지금까지 해온 앱들은 너무 무거움)
- 진짜 기회는 **옵션 A**(1화면 1경기) 또는 **옵션 B**(AI 질문) 중 하나
- **절대 모든 걸 다 하려고 하면 안 됨** — theScore를 이길 수 없음. 한 가지에 미쳐야 함.
